#  🐘 문제 상황 및 프로젝트 배경

1️⃣  학업이수가이드는 학교에서 pdf파일을 제공하지만, 250페이지에 모든 과에 대한 정보가 담겨져 있기 때문에<br/>
내가 원하는 정보가 어디에 있는지 잘 몰라서 자신의 과에 대한 정보를 찾는데 꽤 많은 시간이 소요되고, 이해하는데 어렵다!
